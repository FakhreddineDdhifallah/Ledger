# Ledger 💰

A personal money tracking app built with React Native + Expo.

## Stack
- React Native + Expo SDK 54
- React Navigation (Bottom Tabs + Native Stack)
- React Native Paper (Material Design)
- AsyncStorage (offline persistence)

## Screens
- Splash Screen — animated intro
- Dashboard — balance, monthly stats, recent transactions
- Add Transaction — log income & expenses
- History — full transaction list

## Run locally
```bash
npx expo start --tunnel
```
